Суть проекта - это удобная смесь бота и веб эпп
 Данное приложение автоматически напоминает Вам о днях рождениях!
        Уведомление приходит за указанное в Вашей группе количество дней администратору группы,
        затем администратор делает рассылку для остальных пользователей кроме именинника
        Для вступления в группу нужно заполнить дату своего дня рождения в 'редактировании профиля' и нажать enter
для генерации токена-инвайта выберете нужную группу в админ группах и нажмите сгенерировать инвайт
В целом здесь можно создавать группы, вступать в них, покидать их, можно исключать пользователя из групп
Есть списки групп в которых состоите и список групп которые вы администрируете
Есть также ваш профиль где вы можете отредактировать ваш день рождения
При редактировании даты - во всех группах где вы состоите - перешедулерится ваша дата дня рождения
Можно создать группу и обновить группу
При обновлении интервала "за какое время уведомить админа" - в рамках данной группы перешедулярятся даты всех юзеров
При исключении/выхода пользователя из группы - шедулер объект удаляется
За все это отвечает модель GroupNotification и айдишники в шедулере
Вид айдишника у scheduler job store - G +  GroupNotification.id (G1)
В будущем возможно появятся кастомные напоминания с другой моделью поэтому буква модели добавляется в начале
Админа уведомлеят приложение в тг и предлагает ему команды для поздравления каких либо юзеров
Админ вставляет команду в чат бот ему предлагается вставить сообщение для пользователей группы - типа
вот надо поздравить такого то такого то, давайте думать над подарком и это сообщение автоматически
рассылается всем участникам кроме именинника
